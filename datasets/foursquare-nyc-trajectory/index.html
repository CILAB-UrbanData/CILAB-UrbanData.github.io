Foursquare New York City (FS-NYC)

A Foursquare NYC trajectory/check-in dataset stored as point-wise records. Points are grouped by tid (trajectory id) and include location (lat, lon) plus coarse temporal features (day, hour) and a POI category.

Dataset Description

The dataset is stored as a single CSV file (all_latlon.csv). Each row corresponds to a single check-in / point. Trajectories are formed by grouping rows with the same tid.

Besides latitude/longitude, the dataset includes coarse temporal fields (day, hour) plus a discrete POI category id and a label field (as provided by the dataset).
