Foursquare New York City (FS NYC)

FS NYC is a subset of the Foursquare trajectory dataset for New York City. The file all_latlon.csv contains all trajectories (merged from the original train/test files used by the LSTM‑TrajGAN project). Since you may want custom splits, this page only provides the merged file. The dataset can be used as-is without additional pre-processing.

Schema Notes

all_latlon.csv follows the “lat/lon trajectory” representation used by LSTM‑TrajGAN. In many workflows, each row corresponds to a trajectory, and the trajectory is stored as a sequence of latitude/longitude points (often encoded as a list / serialized structure).

Tip

If you paste the first row (or header) of all_latlon.csv, I can turn it into a precise schema table like your Porto page.
