Dataset Description

We have provided an accurate dataset describing a half year (from 01/01/2016 to 30/06/2016) of the pickup and drop-off information for the green and yellow taxis running in the city of New York, in the USA. All trips are map-matched and aggregated into 264 spatial regions, you can get the spatial document in shapefile format from raw webite.

Data Schema

Field Name Data Type Description Example
VendorID int A code indicating the LPEP provider that provided the record. 1 = Creative Mobile Technologies, LLC 2 = Curb Mobility, LLC 6 = Myle Technologies Inc 1
lpep_pickup_datetime datetime The date and time when the meter was engaged. 2016-02-01 00:10:55
lpep_dropoff_datetime datetime The date and time when the meter was disengaged. 2016-02-01 00:35:08
store_and_fwd_flag string This flag indicates whether the trip record was held in vehicle memory before sending to the vendor, aka “store and forward,” because the vehicle did not have a connection to the server. N
weather_condition string Weather condition during measurement Clear, Rain, Snow, Fog
RatecodeID int The final rate code in effect at the end of the trip. 1 = Standard rate 2 = JFK 3 = Newark 4 = Nassau or Westchester 5 = Negotiated fare 6 = Group ride 99 = Null/unknown 1
PULocationID int TLC Taxi Zone in which the taximeter was engaged. 265
DOLocationID int TLC Taxi Zone in which the taximeter was dsengaged. 265
passenger_count int The number of passengers in the vehicle. 1
trip_distance float The elapsed trip distance in miles reported by the taximeter. 15.46
fare_amount float The time-and-distance fare calculated by the meter. For additional information on the following columns, see https://www.nyc.gov/site/tlc/passengers/taxi-fare.page 8.79
extra float Miscellaneous extras and surcharges. 0.5
mta_tax float Tax that is automatically triggered based on the metered rate in use. 0.5
tip_amount float Tip amount – This field is automatically populated for credit card tips. Cash tips are not included. 3.36
tolls_amount float Total amount of all tolls paid in trip. 0.0
improvement_surcharge float Improvement surcharge assessed trips at the flag drop. The improvement surcharge began being levied in 2015. 0.46
total_amount float The total amount charged to passengers. Does not include cash tips. 15.46
payment_type int A numeric code signifying how the passenger paid for the trip. 0 = Flex Fare trip 1 = Credit card 2 = Cash 3 = No charge 4 = Dispute 5 = Unknown 6 = Voided trip 2
trip_type float A code indicating whether the trip was a street-hail or a dispatch that is automatically assigned based on the metered rate in use but can be altered by the driver. 1 = Street-hail 2 = Dispatch 1.0
congestion_surcharge string Total amount collected in trip for NYS congestion surcharge. None
