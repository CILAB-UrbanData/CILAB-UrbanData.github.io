Dataset Description

This dataset contains trajectories that have been map-matched onto the San Francisco road network. The road graph is derived from OpenStreetMap (nodes/edges), and trajectories are represented as sequences of road segments.

The download link contains 5 cities; for this page, you only need sf_data.

sf_data size: 1,070,351,564 bytes (~1.07 GB).
